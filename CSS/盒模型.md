**一个完整的盒模型元素，从外到内由以下构成：**
- 内边距（padding）
- 边框（border）
- 外边距（margin）
- 长和宽（width 和 height）

```css
/* 浏览器页面有默认的边距 */
body {
	margin: 0;
	padding: 0;
}

.andy {
	margin: 20px;
	border: 5px solid black;
	padding: 10px;
	width: 100px;
	height: 50px;
	background-color: aqua;
}
```
